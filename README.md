# Heraldric Coat of Arms

A static medieval-style heraldry studio for composing a coat of arms from historically inspired tinctures, ordinaries, charges, layouts, and a motto.

## Local development

```bash
npm test
npm run build
```

Open `index.html` directly for quick local viewing, or serve `dist/` after a build.

## GitHub Pages deployment

The repository includes a GitHub Actions workflow that tests the app, builds the static files into `dist/`, uploads the artifact, and deploys it to GitHub Pages on pushes to the `work` branch or manual workflow dispatch.
